## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+

### Installation

```bash
# Clone the repository
git clone https://github.com/Kingrane/WEB-FL.git

# Navigate to project
cd WEB-FL

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

## ğŸµ How to Use

1. **Create Tracks** - Click "+ Insert Track" to add new audio channels
2. **Add Notes** - Click on the Piano Roll to add notes
3. **Edit Patterns** - Drag notes to move, resize from the edge
4. **Apply Effects** - Click "Effects" on any track to add Reverb/Delay/Filter
5. **Export** - Click "Export" to download your mix as WAV
